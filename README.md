# rgx

A fresh, high-performance regex engine aiming to surpass PCRE2 while enabling safe, multi-language code execution within patterns. Built in Rust for speed and safety.

- Core crate: `rgx-core`
- CLI: `rgx-cli`
- Future: WASM, Lua, JS executors; language bindings; benchmarks

Repository: https://github.com/rdje/rgx
